Interaction Flow for Tony (Casual User)
Title: Finding National Parks with all the ammenities and activities.
Description: He would like to see if there is cell phone service and possibly a Ranger on site.
Name and Role of Persona: Tony Panthera, Casual User.
Preconditions: Search for our site on his mobile phone.
Postconditions: Tony (casual User)is able to find ammentities and contact info quickly.
Interaction Flow:

    Navigation steps:

    • User navigates to home site of our National Parks page.
    • The URL fetches the homepage for the user.
    • User selects search bar and inputs his park name.
    • User has a filter option, search for parks with ammentities and activities.
    • The server searches the database and provides a list of matching results.
    • User clicks on the page for his park.
    • The server fetches updated information from the database and submits to the page.
    • User is provided with a list on the ammentities, Ranger contact, and other activities in his area.
