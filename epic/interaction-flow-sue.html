Interaction Flow for Power User (Sue)

Title: Favoriting a New Park
Description: A power user wants to favorite a park she recently visited.
Name and Role of Persona: Sue Bradford, power user
Preconditions: An account on our site
Postconditions: Power user has added a park to her favorites list.

Interaction Flow:

  • User navigates to home site of our National Parks page.
  • The URL fetches the homepage for the user.
  • User selects search bar and inputs her park name.
  • The server searches the database and provides a list of matching results.
  • User clicks on the page for her park.
  • The server fetches updated information from the database and submits to the page.
  • User selects favorite button for the park.
  • The server asks if user is sure.
  • User confirms action.
  • Server adds park to User's favorite list.
